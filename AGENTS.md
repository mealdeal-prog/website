# 🤖 Инструкции для ИИ-агентов (AGENTS.md)

Этот файл содержит важные технические нюансы проекта для ИИ-помощников и разработчиков.

## 1. Стилизация (CSS)
**Критически важно:** Проект использует Tailwind CSS, но без процесса сборки (build process).
- **Проблема:** Новые Tailwind-классы, которых нет в текущем `style.css`, не будут работать.
- **Действие:** При создании новых блоков или серьезном изменении дизайна используйте **Plain CSS** (обычный CSS) внутри `<style>` в `index.html`. 
- **Пример:** Футер полностью реализован на чистом CSS для гарантии стабильности отображения.

## 2. Локализация (Языки)
- Все тексты вынесены в `js/main.js` в объект `translations`.
- При добавлении новых текстовых блоков обязательно добавляйте ключи в секции `LT` и `EN`.
- Выбор пользователя сохраняется в `localStorage` под ключом `mealdeal_lang`.

## 3. Изображения
- Основной формат — **WebP**.
- При добавлении новых фото конвертируйте их в WebP (качество ~88).

## 4. Контактная форма
- Работает через **Web3Forms**. 
- Ключ доступа прописан в `js/main.js`. 
- Если письма не доходят, проверьте подтверждение почты на стороне сервиса.

## 5. Deployment
- Сайт хостится на **GitHub Pages**.
