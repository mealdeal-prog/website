# 💡 Подсказка для Константина

Привет! Вот простые способы, как запустить этот сайт на твоем компьютере, чтобы видеть изменения перед тем, как выкладывать их в интернет.

## 1. Самый простой способ (через VS Code)
Если ты пользуешься **Visual Studio Code**:
1. Установи расширение (Extension), которое называется **"Live Server"**.
2. Открой файл `index.html`.
3. Внизу справа нажми кнопку **"Go Live"**.
4. Сайт автоматически откроется в браузере и будет обновляться каждый раз, когда ты сохраняешь изменения в коде.

## 2. Через терминал (если есть Python или Node.js)
Если ты не используешь VS Code, можно запустить сервер через терминал (командную строку) прямо в папке проекта:

*   **Python (обычно уже есть на компьютере):**
    ```bash
    python3 -m http.server 8000
    ```
    Затем в браузере введи: `http://localhost:8000`

*   **Node.js (если установлен):**
    ```bash
    npx serve
    ```

## 🛠️ Как менять информацию?
- **Тексты:** Все тексты (литовские и английские) находятся в файле `js/main.js` в самом начале, в переменной `const translations`.
- **Фотографии:** Просто добавь новое фото в папку `images/` под тем же именем или замени имя файла в `index.html`.
- **SEO:** Если нужно изменить описание сайта для Google, оно находится в верхней части `index.html` (в секции `<head>`).

Удачи! 🌿
